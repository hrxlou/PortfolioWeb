# 🚀 김현의 개발자 포트폴리오 (hrxlou)

성장하는 즐거움을 아는 대학생 개발자, 김현의 포트폴리오입니다. 깔끔한 디자인과 직관적인 사용자 경험을 선호하며, 코드를 통해 일상의 가치를 더하는 것을 목표로 합니다.

## ✨ 주요 기능 및 특징

- **Modern UI/UX**: 다크 모드와 라이트 모드를 모두 지원하며, Glassmorphism과 Framer Motion을 활용한 세련된 애니메이션을 제공합니다.
- **Data-Driven Architecture**: 모든 포트폴리오 데이터는 `portfolioData.ts` 파일 하나로 관리되어 내용 수정 및 유지보수가 매우 간편합니다.
- **Responsive Design**: 데스크톱부터 모바일까지 모든 기기에서 최적화된 화면을 제공합니다.
- **Premium Card Design**: 프로젝트 이미지를 인셋(Inset) 레이아웃으로 배치하고 라운드 테두리와 그림자를 적용하여 가독성과 심미성을 높였습니다.

## 🛠 Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS (Custom Properties), Framer Motion
- **Deployment**: Vercel

## 📂 프로젝트 구조

```text
src/
├── components/     # UI 구성 요소 (Hero, FeaturedProject, PortfolioGrid 등)
├── data/           # 포트폴리오 내용 (portfolioData.ts)
├── App.tsx         # 메인 앱 컴포넌트
└── index.css       # 글로벌 스타일 및 디자인 시스템
```

## 🚀 시작하기

이 프로젝트를 로컬에서 실행해보려면:

```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev
```
